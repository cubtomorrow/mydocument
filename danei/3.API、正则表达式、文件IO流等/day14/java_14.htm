2.4     线程同步

多个线程，共享修改或访问数据，会引起数据访问冲突

 

l  同步锁：

Java中，任何对象上，都有一个同步锁，

l  多个线程争夺同一个对象的锁，谁抢到谁执行，其他线程要等待锁被归还

 

2.4.1 synchronized 同步

synchronized关键字，要求正在执行的线程，必须获得一个指定对象的锁，才能执行；得不到锁，就要等待
